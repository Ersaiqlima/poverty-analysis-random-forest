# Poverty Analysis Using Random Forest Regression

##  Project Overview
This project applies Machine Learning using the Random Forest Regression algorithm to analyze poverty levels across Indonesian regencies/cities based on socio-economic indicators.

The project focuses on identifying important factors related to poverty and building a predictive model using statistical and machine learning approaches.

##  Dataset
The dataset contains socio-economic indicators from Indonesian regencies/cities.

Variables may include:
- Poverty Percentage
- Human Development Index (HDI)
- Unemployment Rate
- Economic Indicators
- Education Indicators
- Other socio-economic variables

##  Methods
This project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Selection
- Random Forest Regression Modeling
- Model Evaluation

##  Machine Learning Model
Algorithm used:
- Random Forest Regression

Evaluation metrics:
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² Score

## Tools & Libraries
Programming Language:
- Python

Libraries:
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn

## Files
- `Project Machine learning.ipynb` → Jupyter Notebook
- `Project Machine learning.pdf` → Final project report

## Results
The Random Forest Regression model was used to analyze and predict poverty levels based on socio-economic indicators and evaluate the relationship between variables affecting poverty.
